Youtube Channel Performance

Objective: Gain valuable insights into the channel's performance by examining key metrics

INTRODUCTION

In this project, I developed a pipeline using the YouTube API to collect data efficiently from YouTube and process it quickly. The data is then visualized in a dashboard on Power BI, allowing for a fast understanding of the channel's performance over the past month. This approach provides flexibility to analyze any YouTube channel or video by simply inputting the corresponding channel ID , enabling insights through key performance indicators (KPIs) on a monthly basis.


Tools :

- Youtube API
- Open AI/Chat GPT API
- SQL : On MySQL Workbench, Power Query and in Python with Magic SQL
- Python
- Power BI

Japanese learning

Objective: In a Netflix serie, verify the Pareto Principle with subtitles.

INTRODUCTION

Learning a new language can be difficult with all the new vocabulary and grammar to assimilate . According to the Pareto Law, also known as 80-20, focusing on 20% of the most used words should lead to 80% of understanding.
Let's analyze the word structure and distribution in a Netflix serie.

Tools:

- Simple web-scraping and CSV reports: sources -> Netflix, dondon.media, wikipedia
- Power Query and Excel
- SQL: On MySQL Workbench and in Python with Magic SQL
- Python
- Power BI

Spanish Electricity

Objective: Study the price fluctuation over different periods to identify a pattern and find the cheapest moment to use electricity.

INTRODUCTION

When I need to use my washing machine, I look at the electricity price of the current day to know the cheapest hour through this website : tarifaluzhora.es

Sometimes, there is a great gap between the min and max values for the euro/kWh.
For instance, if we look at the prices for the 1st of May 2023, the ratio between the min (0,03136) and max (0,16346) was more than 3x.

Per month, I have a consumption of approximately 225 kWh:

- 225*0,03136 = 7 euros
- 225*0,16346 = 36,8 euros

Which is a delta about 30 euros. (In reality, there are other factors to take into account in the final invoice (potential, taxes, othersâ€¦).This value (energy : consumption * euro/kWh) represents a bit more than 50% of the total amount)

Let's see in this project what the data can tell us regarding the prices behavior over the hours, days, months and years.

Tools : Excel, Python, Regulard statistics, Correlation, Two-way Anova, Tukey's HSD
